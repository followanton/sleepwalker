# Roadmap

Where the public developer tools are headed. This is direction, not a dated commitment. The hosted engine and Console move separately.

## Next

- **`sleepwalker login`** with an OAuth or device flow, so you do not have to paste a key.
- **A network timeout in the CLI** so a stalled host can never hang a command.
- **Sharper first-run guidance** after actions, especially serialization and saved runs.

## Soon

- More language clients beyond JavaScript and Python.
- Shell completions for the CLI.
- `--open` to jump from a run straight to its Console view.
- More cookbook recipes: scheduled reports, exports, competitor tracking at scale.

## Exploring

- A richer agent starter kit for MCP clients.
- Templates for common CI and reporting setups.

Have a request? Open an issue. Good additions are CLI usability fixes, public API examples, MCP client setups, and documentation. See [CONTRIBUTING.md](CONTRIBUTING.md).
