# MCP OAuth setup

Use this endpoint in MCP clients that support hosted OAuth:

```text
https://mcp.sleepwalker.ai/mcp
```

The client will open a Sleepwalker authorization screen. Sign in, review the
requested access, and authorize the connection.

OAuth is the recommended hosted connector flow because it does not require you
to copy a long-lived token into the client.

More detail:

```text
https://www.sleepwalker.ai/docs/mcp/setup/
```
